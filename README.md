# ランダムサウンド再生システム (Auto MP3) 利用マニュアル

本ドキュメントは、VALORANTのフラッシュ回避練習等に使用する「ランダムサウンド再生システム」の操作説明書である。

---

## 【1】 かんたん使い方ガイド

このツールは、ゲームのキャラクターの音が、ランダムなタイミングでなる道具である。

### ダウンロード方法
1. 右上の緑のCodeから、Download Zipを押す。
2. 解凍後、index.htmlを開く。

### スタート
1. 画面の左側にある、緑色の **「開始」** ボタンをおす。
2. その下の黒い四角に「判定待ち...」と出たら準備OK。
3. そのまま待っていると、フラッシュ音がなる。
4. 音がなり終わると、また次の音を待つ状態になる。

### ストップ
赤色の **「停止」** ボタンをおす。

### 調整したいとき
* **音が大きすぎるとき**
    * 「音量」の横にあるつまみを動かして調整する。
* **音がなるまでの時間を変えたいとき**
    * 「間隔」のつまみを動かす。数字を大きくすると、次の音がなるまでの待ち時間が長くなる。

## 【2】 詳細機能・設定マニュアル

### 1. 概要と動作要件
本システムは、設定された確率ロジックに基づき、登録された音声ファイル（VALORANTのフラッシュスキル音など）を不規則なタイミングで再生するWebアプリケーションである。

> **⚠️ 重要：ファイル配置の前提**
> 本システムを使用するには、HTMLファイルと同一階層に **`sound`** という名称のフォルダが必要である。再生対象となる `.mp3` ファイルは、必ずこのフォルダ内に格納されていなければならない。ブラウザのセキュリティ制約上、任意の場所にあるファイルは再生できない仕様である。

### 2. 再生コントロール（画面左側）
基本的な再生制御と、確率エンジンの設定を行うエリアである。

#### 基本操作
* **再生グループ**
    * 再生対象とする音声ファイルのグループを選択する。グループの編集は右側の設定エリアで行う。
* **前回と同じ音の確率を下げる**
    * 有効にすると、直前に再生されたファイルが連続して抽選される確率を50%低下させる。特定の音が連続することを防ぎたい場合に有効である。
* **音量 / 間隔**
    * マスターボリュームと、抽選を行う時間間隔（0.2秒～5.0秒）を設定する。「間隔」ごとに確率判定が行われるため、ここを短くすると高頻度で判定が発生する。

#### 確率詳細設定
デフォルトでは隠れているが、展開することで以下の挙動設定が可能である。

* **確率増加モード**
    * 判定に外れた（音が鳴らなかった）場合、次回の当選確率をどのように上昇させるかを決定する。
    * **等速 (Linear):** 「増加量」で指定した数値(%)を加算していく。（例: 10% → 20% → 30%...）
    * **指数 (Exponential):** 「倍率」で指定した数値を乗算していく。（例: 10% → 15% → 22.5%...）。急激に発生確率を高めたい場合に適している。
* **初期確率**
    * 再生開始時、および音が鳴った直後にリセットされる基準の当選確率である。

### 3. システム設定・ファイル管理（画面右側）

#### 音声ファイル登録
システム上で認識させるファイル名を管理する。

* **登録手順**
    * テキストボックスにファイル名を入力（拡張子不要）して「追加」を押すか、「参照」ボタンからPC内のファイル名を取得して登録する。
* **注意点**
    * ここでの登録はあくまで「リストへの追加」であり、ファイルのアップロードではない。前述の通り、実際のファイルは手動で `sound` フォルダへ配置する必要がある。

#### グループ設定
登録済みのファイルを任意のグループ（例：「イニシエーター」「デュエリスト」等）に分類する機能である。

1. 「編集するグループを選択」プルダウンから対象を選ぶか、新規名を入力して作成する。
2. グループ編集エリアが表示されるため、そのグループで再生させたいファイルにチェックを入れる。
3. **「グループ内容を保存」** ボタンを押下して確定させる。

#### 設定ファイルの保存・読込
現在の登録リスト、グループ設定、確率設定などはブラウザのローカルストレージに保存されるが、JSONファイルとして書き出すことも可能である。

* **エクスポート:** 現在の設定をファイルとしてダウンロードする。バックアップ用途に使用する。
* **インポート:** 以前保存した設定ファイルを読み込み、環境を復元する。
